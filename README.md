# Jin.gpt.ai-greatness

## 소개
이 프로젝트는 단순한 직원 명부 관리를 넘어, 데이터 기반의 체계적인 인사 관리와 효율적인 실무 프로세스 구축을 위해 개발된 종합 HR 관리 프로그램입니다. 실무자의 반복적인 행정 업무를 최소화하고, 조직 내 흩어져 있는 다양한 인적 자원 데이터를 한곳으로 통합하여 전략적이고 객관적인 인사 의사결정을 돕는 것을 목표로 합니다.

## 주요 기능
- **통합 인적 자원 관리:** 임직원의 기본 인적 사항부터 소속 부서, 직급, 발령 이력 및 연도별 인사 평가 등급까지 직원 생애주기 전반에 걸친 데이터를 안전하게 기록하고 추적합니다.
- **SQL 기반 인사 데이터 분석(HR Analytics):** 부서별 성과 지표, 연도별 이직률 추이, 직무별 인건비 비중 등 복잡한 인사 데이터를 다각도로 분석하여 직관적인 통계 화면으로 제공합니다.
- **AI 기술을 활용한 채용 지원:** 신규 인재 채용 시, 지원자의 이력서 및 자기소개서에서 핵심 역량 키워드를 자동으로 추출하고 포지션별 직무 적합도를 분석하여 스크리닝 시간을 단축합니다.
- **근태 및 보상 프로세스 간소화:** 유연근무제, 연차 사용 내역, 출퇴근 기록을 실시간으로 연동하고, 이를 바탕으로 기초적인 급여 정산 및 근태 통계 데이터를 자동 산출합니다.

## 사용 방법
1. **소스 코드 다운로드:** 터미널을 열고 GitHub 저장소에서 최신 버전의 프로젝트 코드를 로컬 환경으로 복제합니다. (`git clone https://github.com/your-username/NextGen-HR.git`)
2. **환경 변수 및 데이터베이스 설정:** 프로젝트 폴더 내의 `.env.example` 파일을 참고하여 `.env` 파일을 생성한 뒤, 로컬 데이터베이스 연결 정보 및 API 키 등을 기입하여 초기 환경을 구성합니다.
3. **패키지 설치 및 프로그램 실행:** 사용 중인 언어 환경에 맞추어 필수 의존성 패키지를 설치한 후(예: `npm install` 또는 `pip install -r requirements.txt`), 실행 명령어(예: `npm run start`)를 입력하여 로컬 서버를 구동합니다. 이후 웹 브라우저를 통해 제공된 로컬 주소로 접속합니다.
